
# zhuyin-captcha

Captcha generator with [Zhuyin](https://en.wikipedia.org/wiki/Bopomofo) (Bopomofo) characters.

## Run the generator server

```shell
$ node server
```

Then you can get a generator by `localhost:3000/test`.
